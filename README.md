# 🌟 Overview

Diana AI Chat is an open-source AI assistant platform designed to provide intelligent conversations, information retrieval, and future educational experiences through a scalable and extensible architecture.

The project aims to create a next-generation AI ecosystem where users can communicate naturally with an intelligent assistant while accessing advanced capabilities such as language learning, search-enhanced responses, personalized assistance, and future AI-powered tools.

Unlike traditional chatbot projects, Diana AI Chat is being developed with long-term scalability and continuous innovation in mind.

# 🚀 Getting Started

## Prerequisites

* Node.js
* PostgreSQL
* npm or yarn

## Installation

```bash
git clone https://github.com/asadbek1877/diana-ai-chat.git

cd diana-ai-chat

npm install
```

## Environment Variables

Create a .env file:

```env
DATABASE_URL=
BOT_TOKEN=
PORT=
JWT_SECRET=
```

## Run Development Server

```bash
npm run dev
```

## Production

```bash
npm run build
npm start
```
